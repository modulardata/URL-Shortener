Description

In this project, we are going to build a simple URL shortener. You may have heard of bit.ly before. It is a good example of a URL shortener. URL shorteners reduce the length of long URLs. Users are redirected to the original URL when they use the short version of a URL. Usually, you need a backend service to create a URL shortener, but we will mimic the functionality with JavaScript.

Let's start by setting up the page with HTML tags.

First, put the first-level heading that says URL Shortener, so we know where we are. Second, add the input with the ID of input-url with the placeholder Enter a url. Next, add a button with the ID of button-create with the text Create. Lastly, you need to list the URLs â€” create an ordered list with the ID of list-url.

Once the page is set up, the functionality is as follows:

  • After entering a URL and clicking on the Create button, generate a random short URL;

  • The short URL should start with localhost/ and the random character set should be 5 characters long after localhost;

  • Before creating the short URL, check whether the given URL is a valid one, such as: https://example.com/123. If not, show a paragraph saying Please enter a valid url in the body. You can use a regular expression pattern to check the validity;

  • The short URL is a link that redirects to the original link in a new browser tab. So, use the original link as the href of the new short link;

  • Show the original URL next to the short link as a text;

  • Append the short link as a list item to the ordered list: list-url.

You can use the IDs we've provided to select the elements.

Objectives

Your page should:

  • Contain the first-level heading with the text URL Shortener;

  • Contain input with the ID of input-url and the placeholder Enter a url;

  • Contain a button with the ID of button-create and the text Create;

  • Contain an ordered list with the ID of list-url;

  • Display the new short link from inside the list-url;

  • Contain a paragraph with text Please enter a valid url if the entered URL is invalid.
