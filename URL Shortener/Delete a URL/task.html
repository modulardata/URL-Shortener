Description

Now, we can create short URLs and count the clicks, let's add a way to delete them.

First, add a new button element in the HTML document with the ID of button-delete and the text Delete.

The delete action should be as follows:

  • If users did not enter any input, after clicking the delete button, remove all the items inside the ordered list;
  • Users should be able to delete any short URL if they enter the matching short URL or the original URL;
  • If it matches more than one URL, it should delete all the matched items.

You can use the provided IDs to select the elements.

Objectives

Your page should:

  • Contain a button with the ID of button-delete and the text Delete;
  • Remove all the list items if there is no input;
  • Remove the list items matched with the input.
