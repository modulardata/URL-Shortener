Description

Most URL shorteners save the click counts. It can be essential to know how many times the link got clicked by users. In this stage, we will implement a similar click counter, although we won't save it anywhere.

After creating a short URL, add a new span element with the text Clicks: inside the list item for the click count. The click count should start with zero and increase every time the user clicks on the short link. Since the link should open in another tab, you should be able to see the increased counts. Each new link should have its own click count.

Objectives

Your page should:

  • Contain a span inside a list item with the text Clicks:

  • Display the click counts in the span ;

  • Increase the click counts when clicked on the short URL.
