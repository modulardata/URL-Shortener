Description

The last functionality is to edit the random part of the short URL.

To do that, we will add a new button with the text Edit after the click count span in the list item.

Edit action should work like so:

  • After clicking the edit button, you should remove the link from the list item and add a new input instead;
  • The input should have the previously created random string as the initial value;
  • The edit button's text should also be Save after clicking it;
  • After changing a random string with the new user input and clicking the save button, the link should be created again with the entered value while removing the input, and
  • Everything should work the same with the link.
Objectives

Your page should:

  • Contain a button inside a list item with the textEdit;
  • Replace the link with an input inside the list item;
  • Have the input with an initial value of the random string, created previously;
  • Replace the button text Edit with Save after the first click, and after the second click it should be Edit again;
  • Contain the link again after saving the change.
